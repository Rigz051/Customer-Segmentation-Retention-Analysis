# Customer Segmentation & Retention Analysis

## About This Project
A machine learning project that predicts customer churn 
for a telecom company using real data of 7,043 customers.

## What This Project Does
- Predicts which customers are likely to leave
- Segments customers into risk groups
- Suggests retention offers
- Shows business impact ($1.88M revenue at risk)
- Interactive dashboard with charts

## Tools Used
- Python
- Pandas
- Scikit-learn (Logistic Regression)
- Matplotlib
- Google Colab

## Results
- Model Accuracy: 79%
- High Risk Customers Found: 2,424
- Revenue at Risk: $1,883,113
- Dataset: 7,043 customers

## Dataset
WA Telco Customer Churn Dataset from Kaggle
